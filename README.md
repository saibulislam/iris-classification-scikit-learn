# Iris Classification using Scikit-Learn

## Overview

This project is part of the **Soft Nexis Technology Virtual Internship Programme**.

The objective of this project is to build and evaluate Machine Learning classification models using the Iris Flower dataset.

## Dataset

- **Dataset:** Iris Flower Dataset
- **Source:** Scikit-learn (`load_iris()`)

## Tasks Performed

- Loaded the Iris dataset
- Prepared features and target labels
- Split the dataset into training and testing sets
- Built a Decision Tree Classifier
- Made predictions on test data
- Evaluated the model using:
  - Accuracy Score
  - Classification Report
  - Confusion Matrix
- Visualized the Decision Tree
- Compared training and testing accuracy for different tree depths
- Built a Random Forest Classifier
- Performed Cross Validation
- Visualized Feature Importance
- Compared multiple Machine Learning models
- Performed Hyperparameter Tuning using GridSearchCV
- Generated the final evaluation report

1. Install the required libraries:

```
pip install -r requirements.txt
```

2. Open the notebook:

```
Iris_Classification_Full.ipynb
```

3. Run all cells from top to bottom.
